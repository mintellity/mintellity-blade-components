# Icon
[Boxicons](https://boxicons.com/)

```php
<x-mint::icon name="arrow-left" />
```

## Attributes:

| Attribute | Description                                                    |
|-----------|----------------------------------------------------------------|
| name      | Name of the Boxicons without the "bx", "bxs" or "bxl" - prefix |

## Optional Attributes:

| Attribute | Description                              | Default | Value                                |
|-----------|------------------------------------------|---------|--------------------------------------|
| color     | Name of the boxicons color without "bx-" | NULL    | `primary` or other valid color name  |
| size      | Name of the boxicons size without "bx-"  | NULL    | `xs` `sm` `md` `lg`                  |
| type      | Filled solid icons / Brand icons         | NULL    | `solid` `brand`                      |
| rotate    |                                          | NULL    | `90` `180` `270`                     |
| border    |                                          | NULL    | `square` `circle`                    |
| animation |                                          | NULL    | `spin` or other valid animation name |
